
# AI導入・システム開発まわりの技術相談

AI導入、生成AI活用、業務システム開発に関する
**要件整理・PoC設計・見積レビュー・開発PM支援**を行っています。

AIエンジニアとして機械学習・自然言語処理・画像解析・生成AIを使った開発案件に関わり、現在は技術が分かるコンサルとして、AI導入や業務システム化の初期検討から開発推進まで支援しています。

## 相談いただけること

| 相談内容                     | 支援できること                            |
| ------------------------ | ---------------------------------- |
| AIを使いたいが、何から始めればよいか分からない | 業務課題の整理、AI活用余地の検討、PoCテーマの設計        |
| 開発会社に相談する前に要件を整理したい      | 業務フロー、必要機能、優先順位、開発範囲の整理            |
| 見積や提案内容が妥当か判断したい         | 技術的な実現性、工数感、リスク、過不足のレビュー           |
| PoCを進めたい                 | 検証目的、評価方法、必要データ、開発範囲の整理            |
| 生成AIを業務に使いたい             | ChatGPT/LLM活用案の整理、プロンプト設計、業務組み込み検討 |
| 技術者と非エンジニアの間で話が噛み合わない    | 技術内容を整理し、関係者が判断できる形に翻訳             |

## 対応しやすい相談

* AIや生成AIを業務に使えるか確認したい
* 顧客からAI活用の相談を受けたが、技術的な判断が難しい
* 開発会社に依頼する前に、要件を整理したい
* 見積や提案内容に違和感がある
* PoCの進め方を決めたい
* 技術者とビジネス側の間に入って整理してほしい
